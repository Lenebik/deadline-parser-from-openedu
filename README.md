# Openedu Deadline Extractor

Create a folder named "misc" in the project's root directory, then create `credentials.json` in mentioned folder with the following structure:

    {
        "moodle": {
            "username": "XXXXXXX",
            "password": "XXXXXXX"
        },
        "chrome": {
            "chrome_profile": "XXXXXXX"
        }
    }

## Dependencies
- Python 3.6+
- Selenium
