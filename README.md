# Openedu Deadline Extractor

Create a `credentials.json` file in the project's root directory with the following structure:

    {
        "moodle": {
            "username": "XXXXXXX",
            "password": "XXXXXXX"
        },
        "chrome": {
            "chrome_profile": "XXXXXXX"
        }
    }

## Dependencies
- Python 3.6+
- Selenium
